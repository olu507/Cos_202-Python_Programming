Mathematical Calculator

This project is a Python mathematical calculator designed to perform basic arithmetic operations such as addition, subtraction, multiplication, division, modulus, and power. It accepts two numbers from the user, performs the selected operation, and displays the result. The program demonstrates the use of Python's input, output, and selection statements

Personal Pocket CGPA Calculator

This project is a Python program that calculates a student's CGPA based on course scores and credit units. It accepts course details from the user, determines the appropriate grades and grade points, calculates the CGPA, and displays the final result along with the student's class of degree. The program demonstrates the practical use of Python in solving academic calculations.
