Personal Pocket CGPA Calculator (PPC)

Project Title

Personal Pocket CGPA Calculator (PPC)

Aim

The aim of this project is to develop a mathematical calculator using Python that performs basic arithmetic operations such as addition, subtraction, multiplication, division, modulus, and exponentiation. The program is designed using Python functions and selection control statements ("if", "elif", and "else") to allow users to choose and execute different mathematical operations through a simple menu-driven interface.

Objectives

To accept numerical input from the user.

To provide a menu that allows the user to select different mathematical operations.

To perform basic arithmetic operations such as addition, subtraction, multiplication, division, modulus, and exponentiation.

To apply Python's selection control statements (if, elif, and else) to execute the selected operation.

To display the correct result of the chosen mathematical operation.

To handle invalid menu selections and division-by-zero errors appropriately.

Program Description

The Mathematical Calculator is a console-based application developed in Python. The program presents the user with a menu of mathematical operations, including addition, subtraction, multiplication, division, modulus, and exponentiation. The user selects the desired operation and enters the required numerical values.
Using Python's selection control statements ("if", "elif", and "else"), the program identifies the selected operation and performs the corresponding calculation. It also includes error handling for invalid menu selections and division by zero to ensure reliable execution.
After processing the user's input, the program displays the result of the selected mathematical operation. The application continues to run until the user chooses the exit option, making it a simple and interactive tool for performing basic mathematical calculations.

Algorithm

1. Start the program.
2. Display the calculator menu showing the available operations:
   - Addition
   - Subtraction
   - Multiplication
   - Division
   - Modulus
   - Power
   - Clear
   - Exit
3. Prompt the user to select an operation.
4. If the user selects Exit, display a closing message and terminate the program.
5. If the user selects Clear, display a confirmation message and return to the main menu.
6. If the user selects a valid mathematical operation:
   - Prompt the user to enter the first number.
   - Prompt the user to enter the second number.
   - Use "if", "elif", and "else" statements to determine the selected operation.
   - Perform the corresponding calculation.
   - If division is selected and the second number is zero, display an error message.
   - Display the result of the calculation.
7. If the user enters an invalid menu option, display an "Invalid Choice" message.
8. Repeat the process until the user chooses to exit the program.
9. End the program.
